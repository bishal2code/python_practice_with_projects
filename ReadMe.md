# Data Structure Practice in Python 🐍

This folder contains Python programs and exercises focused on **data structures**.  
It serves as a collection of practice codes for learning and mastering lists, tuples, sets, dictionaries, stacks, queues, trees, and more.

---

## 📂 Contents

- **list_examples.py** → Practice with Python lists (append, pop, slicing)
- **tuple_examples.py** → Immutable collections and usage
- **set_examples.py** → Unique elements, union, intersection
- **dict_examples.py** → Key-value pairs, counting, lookups
- **stack_queue.py** → Implementation of stack (LIFO) and queue (FIFO)
- **matrix_addition.py** → Adding two matrices
- **log_analyzer.py** → Counting log levels and extracting unique messages
- **tree_graph.py** → Basic tree and graph structures (if included)
